Instructions to run automation:

- Python must be installed (preferred in version 3.7.X)
- Python libs must be installed: Selenium, Behave and Hamcrest

pip install selenium
pip install behave
pip install PyHamcrest

- Import the project into a code editor of your choice
- Access the project root through the terminal of your Operating System, and execute the command 'behave', or if you prefer 'behave --tags = desiredTag' to execute a specific feature.
